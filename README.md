# 🌐 Portfólio Pessoal - Vinicius Luchtenberg

## 📖 Descrição
Este projeto consiste em um website de portfólio pessoal desenvolvido com o objetivo de apresentar informações sobre Vinicius Luchtenberg Pagnussat, minhas habilidades e projetos. O site foi construído utilizando tecnologias básicas de desenvolvimento web, com foco em organização, responsividade e interatividade.

---

## 🎯 Objetivo
O principal objetivo deste projeto é aplicar na prática os conhecimentos adquiridos durante o primeiro semestre do curso de Análise e Desenvolvimento de Sistemas, desenvolvendo um site funcional e visualmente agradável.

---

## 🎓 Disciplina
Projeto acadêmico desenvolvido para a disciplina do curso de **Análise e Desenvolvimento de Sistemas (ADS)**.

- 👨‍🏫 Professor: William Hubner  
- 📅 Semestre: 1º semestre  

---

## 🛠️ Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript
- APIs  
- Google Fonts  

---

## 💻 Funcionalidades
- Navegação entre seções com **scroll suave**
- Estrutura de portfólio pessoal
- Seção "Sobre mim"
- Integração com APIs (Consulta de CEP, CNPJ e Clima)
- Área de contato com links para redes sociais

---

## 🧠 Como foi desenvolvido
O projeto foi desenvolvido utilizando HTML para a estrutura da página, CSS para estilização e layout, e JavaScript para interatividade, como a navegação suave entre seções e consumo de APIs externas.

A organização do código foi feita de forma semântica, separando corretamente as seções do site (Home, Sobre, Projetos e Contato), visando boas práticas de desenvolvimento.
